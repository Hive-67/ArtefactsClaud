100 % Local OFFLINE
Le programme fonctionne directement sur votre appareil (tablette, ordinateur ou téléphone), sans internet ni serveur. Vos données ne quittent jamais votre machine.

Pour échanger vos données, utilisez un espace de partage auquel vous avez accès toutes les deux. Créez-y deux sous-dossiers :

La routine quotidienne
1
Votre sauvegarde — Après chaque utilisation, cliquez sur la 💾 disquette. Téléchargez votre fichier .json et placez-le dans votre dossier sur le Drive.
2
La synchronisation (en fin de journée) — Allez dans le dossier de votre collègue, ouvrez son dernier .json. Copiez tout le texte. Dans l'app : 💾 Sauvegarde › Import JSON, collez et cliquez Importer.
✨ Fusion automatique — Le programme intègre automatiquement les nouveautés de votre collègue sans écraser les vôtres !
