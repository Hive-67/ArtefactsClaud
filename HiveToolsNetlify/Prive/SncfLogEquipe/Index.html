💾 Sauvegarde
📤
Exporter
C'est ta sauvegarde personnelle. Le fichier .json téléchargé contient toutes tes données — conserve-le précieusement. Tu pourras t'en servir plus tard pour restaurer exactement cet état, sur cet appareil ou sur un autre.
📥
Importer
Sélectionne un fichier .json créé via « Exporter », puis valide. Toutes les données qu'il contient sont rechargées automatiquement pour retrouver l'état exact de la sauvegarde au moment de sa création. Fusionner combine avec les données actuelles, Remplacer repart uniquement avec son contenu.
Partager
Encode l'état actuel dans un lien. Copie ce lien et ouvre-le sur un autre appareil (téléphone ↔ PC) pour y retrouver instantanément les mêmes données — fonctionne même sur iPhone, sans fichier ni téléchargement.
♻️
Réinitialiser
Efface toutes les données saisies et remet l'application dans sa configuration d'origine. Action irréversible — à utiliser pour repartir de zéro.
